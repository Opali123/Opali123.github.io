vue简明教程

前端框架

安装

  • 确保当前电脑安装了git和node.js
  • 打开命令行，执行 npm install hexo-cli -g

使用

  1. 创建一个文件夹比如叫demo
  2. 打开cmd命令行工具，执行hexo init .
  3. 执行npm install
  4. 执行hexo server
  5. 打开浏览器访问http://localhost:4000

如何写新的文章

  • 所有的文章都保存在source_posts下面，是以markdown格式的，最终会被hexo渲染成html文件

如何把当前的博客上传到github上面

  1. 执行hexo generate
  2. 打开wamp的www目录(如果wamp坏掉了，可以考虑使用phpstudy)
  3. 把public文件夹里面所有的文件全部拷贝到www根目录(一定要使用根目录)
  4. 通过浏览器访问，确定没有样式上的问题
  5. 打开你的github，假如你的用户名是叫eric,则创建一个叫eric.github.io的仓库
  6. 在www根目录执行git init
  7. 在www根目录执行git remote add origin 仓库地址
  8. 在www根目录执行git push -u origin master
  9. 当push成功之后，访问http://用户名.github.io

如何换主题

  1. 去主题市场找到喜欢的主题
  2. 把当前这个主题下载下来解压放到themes目录里面
  3. 修改config.yaml文件中的theme配置项 yaml theme: 你找到的主题的名字
  4. 执行hexo serve,查看是否切换主题成功
  5. 注意：有的主题有问题，这时候可以换一个主题尝试一下
